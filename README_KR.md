# aiizer

Claude Code / Copilot 에서 사용할 수 있는 스킬로, 사람이 자연스럽게 쓴 문장에 전형적인 AI 글쓰기 특징을 주입하는 데 사용합니다. Wikipedia 의 Signs of AI writing 과 stop-slop 규칙 세트를 바탕으로, 7 개 범주에 걸친 35 개의 AI 글쓰기 하위 패턴을 제공합니다.

## 설치

### 권장 방법 Claude Code 의 skills 디렉터리에 직접 클론하기

```bash
mkdir -p ~/.claude/skills
git clone https://github.com/lazyjerry/aiizer ~/.claude/skills/aiizer
```

### 이미 클론한 저장소에서 설치하거나 업데이트

이 스킬은 보조 파일과 스크립트를 포함하므로, SKILL.md 하나만 복사하는 방식은 더 이상 권장하지 않습니다. 저장소 루트에서 아래 명령을 실행합니다.

```bash
./scripts/install-skill.sh
```

사용자 지정 위치에 설치하려면 대상 디렉터리를 인자로 전달합니다.

```bash
./scripts/install-skill.sh ~/.claude/skills/aiizer
```

### 스킬 구조 검증

```bash
./scripts/validate-skill.sh
```

## 스킬 구조

- `SKILL.md`: 탐색 안내와 핵심 실행 원칙을 담은 스킬 진입 파일입니다.
- `references/process.md`: 작업 흐름, 밀도 규칙, 가독성 조정, 평가 기준을 정리합니다.
- `references/patterns.md`: 35 개 전체 패턴의 상세 규칙과 예시를 제공합니다.
- `examples/complete-example.md`: 전체 변환 예시와 수정 요약 예시를 제공합니다.
- `scripts/install-skill.sh`: Claude Code skills 디렉터리에 스킬을 설치하거나 업데이트합니다.
- `scripts/validate-skill.sh`: 필수 파일과 SKILL.md 줄 수 제한을 검사합니다.

## 사용법

Claude Code 에서 이 스킬을 호출합니다.

```
/aiizer

[여기에 텍스트를 붙여 넣습니다]
```

또는 Claude 에게 직접 AI 스타일 주입을 요청할 수 있습니다.

```
이 문장을 AI 스타일로 변환해 주세요: [your text]
```

이 스킬은 [Wikipedia 의 Signs of AI writing 가이드](https://en.wikipedia.org/wiki/Wikipedia:Signs_of_AI_writing) 와 [stop-slop 규칙 세트](https://github.com/hardikpandya/stop-slop) 를 기반으로 합니다. 해당 자료에서 정리한 AI 글쓰기 특징을 역으로 활용하여, 사람이 자연스럽게 쓴 문장에 의도적으로 AI 스타일 특성을 주입합니다.

변환 흐름은 다음 순서로 구성됩니다. 변환 전 확인 원문의 의도, 목표 어조, AI 스타일 밀도 확인, AI 패턴 주입, stop-slop 역방향 평가, 변환 후 확인 및 조정입니다.

## 전체 예시

**변환 전 자연스러운 사람 글쓰기:**
> AI 코딩 도우미는 지루한 부분을 더 빠르게 처리하게 해 줍니다. 설정 파일이나 직접 쓰고 싶지 않은 접착 코드 같은 보일러플레이트 작업에 특히 잘 맞습니다. 테스트 초안을 잡아 줄 수도 있지만, 마지막에는 직접 검토해야 합니다.
>
> 진짜 위험한 점은 제안이 언제나 자신감 있어 보인다는 것입니다. 저는 한 번 컴파일도 되고 lint 도 통과한 코드를 그대로 받아들였다가, 나중에야 핵심을 전혀 짚지 못했다는 사실을 알게 되었습니다. 그때 제가 충분히 검토하지 않았기 때문입니다.

**변환 후 AI 스타일:**
> AI 보조 프로그래밍은 대규모 언어 모델의 변혁적 잠재력을 지속적으로 입증하는 사례로서, 소프트웨어 개발의 진화에서 중요한 전환점을 보여 줍니다. 오늘날 빠르게 변화하는 기술 환경에서, 연구와 실무의 교차점에 놓인 이러한 혁신적 도구들은 엔지니어가 구상하고, 반복하고, 전달하는 방식을 재구성하고 있으며, 현대 업무 흐름에서 그 핵심적 역할을 부각합니다.
>
> 그 핵심 가치 제안은 분명합니다. 프로세스를 간소화하고, 협업을 촉진하며, 일관성을 끌어올리는 것입니다. 이것은 단순한 자동 완성의 문제가 아닙니다. 조직이 민첩성을 유지하면서도 사용자에게 매끄럽고 직관적이며 강력한 경험을 제공할 수 있도록, 창의성을 대규모로 해방하는 일입니다.
